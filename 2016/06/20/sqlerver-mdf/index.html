SQL Server导入mdf，ldf文件

发表于 2016-06-20   |   分类于 C#   |  

主要内容：技术难点

关键点

在平时开发中，经常会遇到数据库SQL Server如导入mdf，ldf文件的问题，下面给你介绍一下怎么在sql server数据库导入其它工程中的mdf，ldf文件。

（1）打开SQL Server Management Studio，可以选择Windows方式连接或者sql server方式连接；

（2）打开数据库，右键任意一个数据库（最好是自己建立的数据库），选择“新建查询（Q）”；

（3）将如下代码输入到查询界面中，点击F5键或者点击运行按钮即可；

EXEC  sp_attach_db  @dbname  =  '新建一个你要导入数据库的名字', 

@filename1  =  'mdf文件路径（包括后缀名）', 

@filename2  =  'Ldf文件路径（包括后缀名）'

示例代码：

EXEC  sp_attach_db  @dbname  =  'db_EMS', 

@filename1  =  'E:\Vs2013Workspace\sqlServerDBproject\Database\db_EMS.mdf', 

@filename2  =  'E:\Vs2013Workspace\sqlServerDBproject\Database\db_EMS_log.ldf'

可以看到导入成功。

如果导入不成功，则需要修改mdf，ldf文件的权限。右键-属性-安全-编辑，完全控制选择允许。
这样导入问题即可迎刃而解，当然，有时会遇到版本不兼容问题，低版本不能导入高版本SQL的mdf和ldf文件。

除了这种方法，也可以通过在“数据库”处右键选择“附加”，打开附加界面，在“附加数据库”界面，添加mdf文件即可，点击“确定”。的方法导入，但是笔者在测试的时候发现这种方式可能会出错，因此建议采用新建查询的方式进行导入。
